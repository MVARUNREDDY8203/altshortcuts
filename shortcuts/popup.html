Alt Shortcuts
Alt + S Jump to Search
Alt + W Jump to Write/Compose
Alt + R Open a new tab to the Right
Alt + L Open a new tab to the Left
Alt + -> Go Forward in History
Alt + <- Go Backward in History
Alt + E Move to the adjacent-right tab
Alt + L Move to the adjacent-left tab
Alt + D Close the current tab
